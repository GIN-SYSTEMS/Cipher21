## 🛡️ Trust & Integrity (Heuristic Notice)

Cipher21 is fully **Open-Source**. Transparency is our core mission.

* **Heuristic Analysis:** Since Go binaries are compiled without expensive digital signatures, some scanners (Bkav, DeepInstinct) may trigger a **False Positive**.
* **Zero-Privilege:** The application requires no administrator rights and only interacts with its local `.db` file.
* **Audit:** We encourage all users to audit the source code or build from scratch for maximum certainty.
